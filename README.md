# 👥 HR Data Analysis Dashboard

## 📌 Overview

This project focuses on analyzing Human Resources data to understand employee behavior, attrition patterns, and workforce distribution.
The dashboard provides insights to support HR decision-making and improve employee retention.


## 🎯 Objectives

* Analyze employee distribution across departments
* Identify attrition trends and key factors
* Evaluate employee performance and satisfaction
* Understand demographics (age, gender, education)
* Provide actionable HR insights


## 🛠️ Tools & Technologies

* Tableau
* Dashboards
* Filters
* Parameters
* Calculated Fields

* 
## 📂 Dataset

The dataset includes:

* Employee demographics
* Department & job roles
* Salary & compensation
* Attrition status
* Performance ratings


## 📊 Dashboard Features

* Employee Count KPI
* Attrition Rate
* Employees by Department
* Attrition by Job Role
* Performance Analysis
